# CourseTracker — 课程打卡进度追踪

一个纯前端的课程打卡进度追踪工具，无需服务器，直接在浏览器打开 HTML 文件即可使用。

目前包含两个版本：

| 文件夹 | 追踪器 | 适用课程格式 |
|--------|--------|-------------|
| `模特徐2026三期/` | ArtStudyTracker | 每节课独立打卡，设有7天补卡窗口 |
| `菲什2026人体/` | FishCourseTracker | 按打卡次分组，每次包含多节课 |

## 功能

- 按状态自动标色：已完成（绿）/ 紧急（黄）/ 已逾期（红）/ 未开始（灰）
- 可视化倒计时方格，直观显示剩余天数
- 勾选完成状态，**自动保存到浏览器缓存（localStorage）**，刷新页面不会丢失

## 进度保存说明

打卡进度保存在**浏览器的 localStorage** 里，不写入文件。这意味着：

- 换浏览器或换设备后，进度不会自动同步
- 清除浏览器缓存会导致进度丢失

建议定期使用**导出备份**功能保存进度文件（JSON），需要时再通过**导入记录**恢复。

## 导出 / 导入

- **导出备份**：将当前打卡进度保存为 JSON 文件，可用于备份或跨设备迁移
- **导入记录**：从 JSON 文件恢复打卡进度

## 用于其他课程

如果你有不同的课程，可以直接把 HTML 文件 + 你的课程表（截图或文字）发给 ChatGPT / Claude 等大语言模型，让它帮你替换课程数据，生成适合你自己课程的版本。

提示词示例：
> 这是我的课程打卡追踪工具（附上 HTML 文件），我有一门新课，课程表如下：[粘贴课程表]。请帮我把 HTML 里的课程数据替换成我的新课程。
